# 数据库集合创建指南

## 方法一：使用 Navicat（推荐）

1. **打开 Navicat for MongoDB**

2. **连接数据库**
   - 主机：localhost
   - 端口：27017
   - 点击"连接"

3. **创建数据库**
   - 右键点击连接 → "新建数据库"
   - 输入数据库名：`jiangjie_museum`

4. **执行集合创建命令**
   - 在新建的数据库中，点击工具栏的"查询"按钮
   - 将 `createCollections.js` 文件的全部内容复制粘贴到查询窗口
   - 点击"运行"按钮执行

5. **验证结果**
   - 展开数据库，应该能看到三个集合：
     - `products` (5 个商品)
     - `volunteeractivities` (3 个活动)
     - `travels` (4 个导览)

## 方法二：使用 MongoDB Shell

1. **打开命令行**
   ```bash
   mongosh
   ```

2. **切换到数据库**
   ```javascript
   use jiangjie_museum
   ```

3. **执行创建命令**
   - 复制 `createCollections.js` 文件内容（不包括注释中的//部分）
   - 在 MongoDB Shell 中粘贴并执行

4. **验证数据**
   ```javascript
   // 查看 products 集合
   db.products.find()
   
   // 查看 volunteeractivities 集合
   db.volunteeractivities.find()
   
   // 查看 travels 集合
   db.travels.find()
   ```

## 方法三：使用后端 seed.js 脚本（最简单）

1. **确保 MongoDB 服务已启动**

2. **在后端目录执行**
   ```bash
   cd backend
   node seed.js
   ```

3. **看到以下输出表示成功**
   ```
   MongoDB连接成功
   商品数据插入成功
   志愿活动数据插入成功
   导览数据插入成功
   所有示例数据插入完成！
   ```
